# Premium Developer Portfolio

بورتفوليو مطور احترافي — **HTML + CSS + JavaScript** فقط.

## التشغيل

### 1. فتح مباشر
افتح `index.html` في المتصفح (Chrome أو Edge).

### 2. Live Server (موصى به)
في Cursor / VS Code: كليك يمين على `index.html` → **Open with Live Server**.

### 3. سيرفر محلي
```bash
cd d:\protofile\protofile\cool-responsive-portfolio-main
npx --yes serve .
```

## هيكل المشروع

```
index.html       ← الصفحة الرئيسية
favicon.svg
css/             ← التنسيقات
js/
  data.js        ← المحتوى (عدّل هنا)
  app.js         ← التفاعلات
  particles.js   ← خلفية الجسيمات
images/          ← صورك
```

## تخصيص المحتوى

عدّل **`js/data.js`**: الاسم، الإيميل، المشاريع، المهارات، الخبرة، الشهادات، الروابط.
